Steps to reproduce

> Clone the repo
> 
> Change directory to `whatsapp-spam-bot`
> 
> Run `npm install`
> 
> Run `npm install pupeteer`
> 
> Change the contact name and the number of times you want to spam a message in index.js file.
> 
> Run `node index.js`
> 
> Scan the QR code from your phone.
>
> See the results : )
